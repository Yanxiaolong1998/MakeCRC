#-----------------------------作为第三方程序调用使用方法针对bin文件--------------------------------------#

注释：调用时使用了4个参数
[argv1]:原文件目录                                      //示例： C:/w.bin
[argv2]:目标目录（为了不破坏原文件）	 //示例： D:/w.bin
[argv3]:计算方式CRC16/CRC32		 //  'crc16'  or  'crc32'   注意是小写
[argv4]:大小端                                            //   '-L'  or  '-S'   注意是大写


完整示例：MakeCRC   C:/w.bin  D:/w-1.bin  crc16  -L
